# Drum Kit – Sound Board

This project is a simple interactive sound board built with vanilla JavaScript.
It allows users to play different sounds by pressing specific keys on the keyboard.

## Overview

Each key is mapped to a sound, and when pressed:

* The corresponding audio is played
* A visual animation is triggered on the associated element

This project focuses on handling keyboard events, manipulating the DOM, and synchronizing audio with UI feedback.

## Features

* Keyboard-driven interaction
* Instant sound playback with replay support
* CSS-based animations on key press
* Clean separation between structure, styles, and logic

## Improvements

Compared to the original version of the exercise:

* Custom sound effects were added
* Styles were modified for a different visual design
* JavaScript was slightly modernized

## Technologies

* HTML5
* CSS3
* JavaScript (ES6)

## How to Use

1. Open the `sounds.html` file in your browser
2. Press the keys shown on screen (A, B, C, D)
3. Each key will trigger a different sound and animation
