# JavaScript30 – Practice Projects

This repository contains my work through the JavaScript30 challenge, focused on improving my core JavaScript skills using vanilla JavaScript.

The goal of this project is to strengthen my understanding of fundamental concepts such as DOM manipulation, events, and application logic by building small, interactive projects.

While following the original structure of the challenge, I aim to add my own modifications and improvements to each project to reinforce learning and make the work more personal.

This repository serves as a record of my progress and a foundation for future work with more advanced tools and frameworks.

Based on the JavaScript30 challenge by Wes Bos.
